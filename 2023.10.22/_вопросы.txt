Данна такая стуктура index.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>DOM API tutirial</title>
</head>
<body>
    <p id="caption">
        Web is not <b>Dead</b>
    </p>
</body>
</html>

1. Создайте js файл script.js и побключите его к index.html

2. В файле script.js создайте переменую которая бы ссылалась по идитификатору на елемент <p id="caption"> ... </p>

3. Переменная 'element' ссылается на елемент  <p id="caption"> ... </p>
   какой рерузльтат будет если 
   console.log(element.innerHTML)
   и 
   console.log(element.innerText)
