
cd 
    Переход в папку(Exm: cd /MyProject)

mkdir
    Создать папку(Exm: MyFolder_1)

echo "Some test" > file_1.txt
    Перезаписывает файл file_1.txt на содержимое "Some test"
    Если такого файла нет то он создается в текущей директории 

ls
    список файлов и папок в данной директории

cat file_1.txt
    Чтение файла и вывод содержимое в консоль

rm file_1.txt
    Удаление файла 'file_1.txt'
