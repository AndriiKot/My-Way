

git commit 
    Откроит текстовый редактор по умолчанию

git commit -m "Some Commit"
    Сделает коммит с заданным текстов в кавычках 

если забыли подготовить 
изменения в файле, который хотели добавить 
в этот коммит, вы можете сделать так:

git commit --amend
    Можно отредактировать последний коммит
    или оставить оставить коммит а произвести 
    изменения в самом проекте
Example:
$ git commit -m 'Some commit'
$ git add file1.txt
$ git commit --amend
