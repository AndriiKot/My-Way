
Посмотреть ЛОКАЛЬНЫЕ ветки
git branch

Посмотреть УДАЛЁННЫЕ ветки
git branch -r
    !!! Без запроса на удалённый сервер !!!

Что-бы посмотреть последние изменения 
нужно сделать
git fetch
git branch -r

Посмотреть все ветки со всеми изменениями
git fetch
git remote prune origin
git branch -a
git branch --all

Удалить все ветки, которых нет во внешнем репозитории 
git remote prune origin
