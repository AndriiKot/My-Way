
git tag
    Выводит все теги в репозитории со всех веток

git tag v1.0.0
    Добавление в локальный репозиторий tag с именем v1.0.0
    В последний коммит

git tag -a v1.5.0 7818b745325936d9eace49098b494f8158b4122b
    Добавление 

git push origin --tags
    Добавляет все теги в удалённый репозиторий

git push origin v1.0.0
    Добавляет tag с именем 'v1.0.0' в удалённый репозиторий

git tag -d v1.0.0
    Удаление локально tag с именем 'v1.0.0' 

git push origin --delete v1.0.0
    Удаление с удаленного репозитория tag с именем 'v1.0.0'
