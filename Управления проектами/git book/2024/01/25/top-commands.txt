// STATUS
cmd:
git status
    Показывает на какой ветки мы находимся
    и если ли какие-то изменения 

сmd:
git status
>
'On branch main'
	Показывает на какой мы ветки находимся

'Untrakced files: '
  test_1.txt
    Показывает изменения в данном файле 

// ADD
cmd:
git add .
    Добавляет все файлы которых было произведёно изменения

git add *
    Добавляет все файлы которых было произведёно изменения

git add test_1.txt
    Добавляет только файл test_1.txt в котором было произведено изменения

// COMMITS
cmd:
git commit -m "Massage to commit"
