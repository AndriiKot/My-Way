ln -s <absolute path> <link name>
	сделать символическую ссылку на файл или директорию

ln  <file name> <duplicated file name>
	сделать дубликат на файл (Дубликат файла - ссылка на
	одного и того же место данных, но с другим именем !!!
	!!! При изменении одного из файлов у которого есть
	дубликаты остальные дубликаты также будут изменяться)
	при удаление одного файла дубликата остальные останутся
