В терминале Ubuntu можно выполнять 
несколько команд последовательно, 
используя символы `;`, `&&` и `||`, 
аналогично Windows Command Prompt. Вот как это работает:

1. **Символ `;`**: Выполняет команды последовательно, 
   независимо от успеха предыдущей команды. Например:

   ```bash
   команда1; команда2; команда3
   ```

2. **Символ `&&`**: Выполняет следующую команду 
   только в том случае, если предыдущая команда 
   завершилась успешно (с кодом возврата 0). Например:

   ```bash
   команда1 && команда2 && команда3
   ```

3. **Символ `||`**: Выполняет следующую 
   команду только в том случае, 
   если предыдущая команда завершилась 
   с ошибкой (с ненулевым кодом возврата). Например:

   ```bash
   команда1 || команда2
   ```

### Пример:
Если вы хотите сначала скопировать файл, 
а затем удалить оригинал, вы можете использовать:

```bash
cp файл_источник.txt файл_назначение.txt && rm файл_источник.txt
```

В этом примере команда `rm` (удаление файла) 
выполнится только в том случае, 
если команда `cp` (копирование файла) завершится успешно.

### Дополнительный пример с использованием `;`:
Если вы хотите выполнить несколько 
команд независимо от успеха предыдущих, вы можете использовать:

```bash
echo "Копирую файл"; cp файл_источник.txt файл_назначение.txt; echo "Файл скопирован"
```
