Чтобы добавить псевдоним (alias) `dev='npm run dev'` 
в командную строку Windows (cmd), следует использовать функционал, 
доступный в PowerShell, так как сама по себе командная 
строка CMD не поддерживает псевдонимы так, как это реализовано в UNIX-системах.

### Способ 2: Использование Batch файла

1. Откройте текстовый редактор (например, Блокнот).
2. Вставьте следующий код:

   ```bat
   @echo off
   npm run dev
   ```

3. Сохраните файл с именем `dev.bat`.
4. Переместите этот файл в папку, 
		которая включена в переменную среды `PATH`, например, 
		`C:\Windows\System32` или другую папку в переменной `PATH`.

Теперь вы сможете просто вводить `dev` в командной строке, 
и он будет запускать `npm run dev`.
Однако, вы можете создать пакетный файл 
для выполнения этой команды или использовать PowerShell. \
Вот как это сделать:

### Способ 1: Использование PowerShell

1. Откройте PowerShell.
2. Чтобы создать псевдоним, введите следующую команду:

   ```powershell
   Set-Alias dev 'npm run dev'
   ```

3. Чтобы этот псевдоним сохранялся даже 
после перезапуска PowerShell, 
добавьте строку выше в файл профиля PowerShell. 
Файл профиля обычно находится по пути 
`C:\Users\<ВашеИмяПользователя>\Documents\PowerShell\Microsoft.PowerShell_profile.ps1`. 
Если его нет, создайте новый.
