
Вы можете использовать командную строку (cmd) в
Windows для перемещения нескольких папок в папку `_paraxe_`. Для этого выполните следующие шаги:

1. Откройте командную строку. Вы можете сделать это, нажав `Win + R`, затем введите `cmd` и нажмите Enter.

2. Перейдите в директорию, где находятся папки. Используйте команду `cd` (change directory).
   Например, если ваши папки находятся в `C:\Users\ВашеИмя\Documents\`, введите:

   ```cmd
   cd C:\Users\ВашеИмя\Documents\
   ```

3. Теперь вы можете использовать команду `move` для перемещения папок.
   Например, если вы хотите переместить папки `folderq` и `folder2` в `_paraxe_`, используйте такую команду:

   ```cmd
   move folderq _paraxe_
   move folder2 _paraxe_
   ```

4. Если папки, которые вы хотите переместить,
   имеют общий префикс (например, `folder1`, `folder2` и т. д.), вы можете использовать символ подстановки `*`. Например:

   ```powershell
   powershell.exe
   move folder* _paraxe_
   ```

Это переместит все папки, начинающиеся с `folder`, в папку `_paraxe_`.

Обратите внимание, что если папка `_paraxe_` не существует,
вам нужно будет ее создать перед перемещением. Вы можете сделать это с помощью команды:

```cmd
mkdir _paraxe_
```

Теперь, выполнив указанные действия,
вы сможете переместить несколько папок в желаемую директорию.
