Да, функцию `acp` можно реализовать в командной строке Windows (cmd), 
 


1. Откройте текстовый редактор (например, Notepad).
2. Вставьте следующий код:

```cmd
@echo off
setlocal

set "source=%~1"
set "destination=%~2"

if "%source%"=="" (
    echo Source directory is not specified.
    exit /b 1
)

if "%destination%"=="" (
    echo Destination directory is not specified.
    exit /b 1
)

robocopy "%source%" "%destination%" /E

endlocal
```

3. Сохраните файл с расширением `.cmd`, например, `acp.cmd`.

Теперь вы можете использовать этот файл в командной строке. Например, чтобы скопировать файлы из директории `..\Array\` в директорию `E:`, выполните:

```cmd
acp ..\Array\ E:\
```

Вы можете сохранить файл `acp.cmd` в любом удобном для вас месте на вашем компьютере, 
но для удобства использования рекомендуется сделать следующее:
(C:\Scripts\acp.cmd)

1. **Сохранить в папке с системными командами**:
   Если вы хотите использовать команду `acp` из любой папки в командной строке, 
   вы можете сохранить файл в папке, которая находится в переменной окружения `PATH`. 
   Например, вы можете сохранить его в папке `C:\Windows\System32`. or (C:\Scripts\)
Однако, будьте осторожны и убедитесь, что у вас есть права на запись в эту папку.

2. **Сохранить в своей пользовательской папке**:
   Вы также можете сохранить файл в папке, которая вам удобна, например, в вашей домашней папке или любой другой папке, и добавить эту папку в переменную окружения `PATH`. Для этого выполните следующие шаги:
   - Создайте папку, например, `C:\Scripts`.
   - Сохраните файл `acp.cmd` в этой папке.
   - Добавьте папку в переменную окружения `PATH`:
     1. Нажмите `Win + R`, введите `sysdm.cpl` и нажмите `Enter`.
     2. Перейдите на вкладку "Дополнительно" и нажмите на кнопку "Переменные среды".
     3. В разделе "Системные переменные" найдите переменную `Path` и выберите "Изменить".
     4. Нажмите "Создать" и введите путь к вашей папке, например, `C:\Scripts`.
     5. Нажмите "ОК", чтобы сохранить изменения.

Теперь вы сможете вызывать вашу функцию `acp` из любой директории в командной строке, 
просто введя `acp` и передав необходимые параметры.
