Tab             Перемещение между вкладками
up, down        Выбор сведущего или предыдущего файла или папки
left, right     Вернуться в начало или в конец данной директории

Shift + right,
Shift + left    Выделить или отменить все файлы от данной позиции курсора до конца

Ctrl + D        Перемещаться по командной строке farManager вправо
Ctrl + S        Перемещаться по командной строке farManager влево


Ctrl + rows     Изменить размер окна
Alt + right     Смотреть имена файлов которые не вместились в колонку
Ctrl  + , -     Изменить размер шрифта
Ctrl + U        Поменять панели местами
Ctrl + P        Убрать не активную панель
Enter           Для архивов войти внутрь архива
Alt + [1..9]    Поиск файла по имени файла символы не водиться если такого файла нет
Alt + [1..9]    *.txt or *keys   Также работает поиск по маске  только для первого найдено совпадения
Shift + F10     Последнее свойство в котором были

Alt + F8        Посмотреть историю команд
Ctrl + A        Редактировать атрибуты
Ctrl + Z        Редактировать описание

Shift + F1      Сжать в архив
Shift + I        Фильтр по расширениям
