0 -> 19 min basic settings

Alt + F1(F2) + Enter    Очень удобно для открытие в противоположной панели тот-же каталог
Tab           переключение между панелями
Alt + F1      сменить диск на левой панели
Alt + F2      сменить диск на правой панели

Alt + F1 ->  Ctrl + 1    Показывает является ли диск жестким или нет при повторном нажатии убирается
                         индикатор жесткого диск (Exm: fixed)
Alt + F1 ->  Ctrl + 3   Название диска
Alt + F1 ->  Ctrl + 4   Тип файловой системы
Alt + F1 ->  Ctrl + 5   показывает свободное место на диске
Alt + F1 ->  Ctrl + 7

Alt + F1 -> 0(Process list) -> F8     убить выбранный процесс

Alt + F2 -> выбрать тот-же диск будет выбрано и тоже место при переходе

Alt + Delete             затереть данные без восстановления

Shift + F9               сохранить настройки

Files highlighting and sort group      Раскраска файлов и группы сортировки (.js .html)

Alt + F9                расширить или с узить окно программы
