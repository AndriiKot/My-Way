	
alt + 1 
    Открывает меню поиска.Для быстрого поиска папок или файлов

alt + F8
    Открывает список команд ранеи введённых 

alt + F1
or 
alt + F2
    Сменить диск(Открыть окно для выбора основных действий)

left arrow
    переход в начала списка файлов и каталогов

right arrow 
    переход в конец списка файлов и каталогов

Tab
    переход на соседнею вкладку
Ctrl + O
    Переход в полноэкранный режим консоли или обратно вернутся в far

Shift + F4
    Создать файл 

Files:
F4
    Открывает файл для редактирования 
F2 
    Сохранить изменения в файле
Esc 
    Выход из файла 
F6
    Переместить файл на соседнею вкладку
Shift + F6
    Переименовать файл или его переместить (переместить с Переименованием)
F5 
    Скопировать файл
F8 
    Удаление файлов
F3, F4
    Открыть файл и посмотреть бинарный код
Shift + F8
    Выбрать кодировку файла

F7
    Создать папку

Folder:
Enter 
    Открывает папку 
Shift + Enter 
    Открытие папки в проводнике
