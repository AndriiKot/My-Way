                                           
JSON(англ. JavaScript Object Notation) - текстовый структурированный формат хранения и обменна даннымиб
основанный на JavaScript. При этом формат независим от JavaScript и может использоваться в любой среде.

Где используется JSON

*  Внутренний формат JavaScript и других языков программирования.

* Формат информационного обмена между сисмемами и их компонентами(RESR,GraphQL и пр.).

* Формат хранения данных некотоых NoSQL БД.

* Нереляционные возможности реляционных БД.

* Хранение и изменения настроек ПО.

...  

JSON-объект
JSON-объект - неупорадоченное множество пар 'ключ-значение'.
Ключ идентифицирует пару, значение несёт смысловую нагрузку.
JSON-объект заключается в фигурные скобки - {} .
Пары 'ключ-значения' отделяются друг от друга запятми
Между ключом и значение ставится знак : 
JSON-объект может быть пустым.


Ключ (key)

* Ключ описывается строкой и заключается в кавычки - "".
* Имя ключа в составе объекта должно быть уникальным.
* Ключ принять именовать английскими словами, строчными буквами.
* Если для имени ключа необходимо испоьзовать несколько слов, то все слова, 
  начиная со второго, принято писать с прописной буквы (например: directorLastName).

Типы значения (value) по ECMA-404

Object
array
number
string
true
false
null

Типы значений
  Простые:
   	 * число (number);
         * строка (string);
	 * литерал (true, false, null).
  Сложные:
         * объект (object);
         * массив (array).

Число(Number)
  * Может быть целым или вещественным.
  * В роли десятичного разделителя используется точка.
  * Допускается экспоненциальная форма представления числа (например, 
    0.12e+5 или 0.25e-5).
  * Число Не заключается в кавычки (иначе оно превращается в строку).

Строка(String)
  * Строка - упорядоченное множество из нуля или более символов юникода.
  * Строка заключается в двойные кавычки.
  
Литералы(true,false,null)
  * не заключены в кавычки

Object
Array
  * массив - упорядоченное множество значений
  * массив заключается в квадратные скобки []
  * значения разделяются запфтми
  * в массив могут входить значения любого типа (число,  сктрока, литерал, объект, массив).
  * Массив может быть пустым, то есть не содержать ни одного значения.
