
Примечание
Можно указывать несколько 
параметров --author и --grep для поиска, 
которые позволят найти коммиты, 
соответствующие любому 
указанному --author и любому указанному --grep шаблону; 
если, применение опции 
--all-match 
будет искать коммиты 
соответствующие всем указанным --grep шаблонам.
