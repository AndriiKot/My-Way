

[attr]
Элемент с атрибутом по имени attr.

[attr=value]
Элемент с именем атрибута attr и значением совпадающим с value.

[attr~=value]
Элемент с именем атрибута attr значением которого является набор слов разделённых пробелами, 
одно из которых в точности равно value

[attr|=value]
Обозначает элемент с именем атрибута attr. Его значение при этом может быть или в точности равно "value" 
или может начинаться с "value" со сразу же следующим "-". 

[attr^=value]
Элемент с именем атрибута attr значение которого начинается с "value"

[attr$=value]
Элемент с именем атрибута attr чьё значение заканчивается на "value"

[attr*=value]
Элемент с именем атрибута attr чьё значение содержит одно или несколько вхождений строки "value" как подстроки.
