grid-repeat

Допустим, вам нужна сетка из 
100 строк одинаковой высоты. 
Вставлять 100 значений по отдельности 
не очень практично. К счастью, есть лучший 
способ — использовать функцию, repeatчтобы указать, 
сколько раз вы хотите, чтобы ваш столбец или строка 
повторялись, после чего следует запятая и значение, 
которое вы хотите повторить.

Вот пример создания сетки из 100 строк, 
каждая строка высотой 50 пикселей.

grid-template-rows: repeat(100, 50px);
