Этот фрагмент создает пять столбцов. 
Ширина первого столбца равна ширине его содержимого, 
второй столбец — 50 пикселей, третий столбец — 10% 
контейнера, а последние два столбца;
оставшееся пространство разделено на три секции: 
две отведены для четвертого столбца и одна для пятого.
