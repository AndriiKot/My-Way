Выровняйте все элементы по вертикали, 
      используя align-items

      
Использование align-items
свойства в контейнере сетки 
установит вертикальное выравнивание 
для всех элементов нашей сетки.

stretch: default
start: выравнивает содержимое слева от ячейки,
center: выравнивает содержимое по центру ячейки,
end: выравнивает содержимое справа от ячейки.
