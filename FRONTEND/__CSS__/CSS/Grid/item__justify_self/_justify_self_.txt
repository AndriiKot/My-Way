Выровняйте элемент по горизонтали, используя justify-self

В CSS Grid содержимое каждого элемента расположено в поле, 
которое называется ячейкой . Вы можете выровнять 
положение содержимого внутри ячейки по горизонтали,
используя justify-self свойство элемента сетки. 
По умолчанию это свойство имеет значение stretch, 
что позволяет содержимому заполнить всю ширину ячейки. Это свойство CSS Grid принимает и другие значения:

stretch: default
start: выравнивает содержимое слева от ячейки,
center: выравнивает содержимое по центру ячейки,
end: выравнивает содержимое справа от ячейки.
