grid-gap

это сокращенное свойство grid-row-gap
двух grid-column-gapпредыдущих задач, 
которое более удобно использовать. 
Если grid-gapимеет одно значение, 
между всеми строками и столбцами будет создан пробел. 
Однако если есть два значения, 
первое будет использоваться для 
установки разрыва между строками, а второе значение — для столбцов.
