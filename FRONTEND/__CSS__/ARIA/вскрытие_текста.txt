Существует распространенный шаблон визуального скрытия текста,
доступный для чтения только программам чтения с экрана.
Этот шаблон предназначен для установки следующих свойств CSS:

position: absolute;
width: 1px;
height: 1px;
padding: 0;
margin: -1px;
overflow: hidden;
clip: rect(0, 0, 0, 0);
white-space: nowrap;
border: 0;
