action - определяет адрес скрипта который обрабатывает данную форму
fieldset - используется в форме для обледенения родственных элементов
legend - заголовок для набора полей форм(идёт сразу за тегом <fieldset>)
label  - метка для элемента <input> так же обеспечивает лучшее взаимодействие со страницей
