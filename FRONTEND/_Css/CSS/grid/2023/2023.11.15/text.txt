.container
    родительский элемент с display: gird 
    настраивает отступы между дочерними элементами (свойство gap)
    может настраивает количество строк и колонок

item
    вложенный в контейнер элемент
    имеет настройки размеров и позиционирования

Gird Line (линия)
    разделительная линия в структуре сетки
    вертикальная и горизонтальная
    отступы gap между ячейками находятся на Grid лини

Grid Cell (ячейка)
    пространство между соседними строками и двумя соседними
    линиями сетки столбцов
    едины "блок" сетки

Grid Track
    пространство между двумя соседними
    линия сетки
    аналог столбца или строки сетки

Grid Area 
    область ячеек внутри сетки
    может состоять из любого количество ячеек
