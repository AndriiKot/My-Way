За год своей деятельности создал много проектов в которых
есть папка node_modules

Решил их все удалить.
Написал скрип который может удалять все заданные папки из заданной директории
Причем папок для удаление может быть заданно несколько

Аргументы для удаление могут браться из ARGV если там нет то скрипт запросит
задать их 

Будут обрабатываться только валидные значения, не валидные будут пропускаться 

Дубликаты в аргументов если таковые есть удаляются

Если скрипту надо удалить заданную папку то он уже на смотрит в глубь.
Это особенно полезно с папками типа node_modules так как в этих папках есть 
свои папки node_modules.

На тестовым проекте скрип отработал корректно
