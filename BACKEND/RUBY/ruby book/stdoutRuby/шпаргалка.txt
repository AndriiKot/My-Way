Чтобы перенаправить вывод в файл в IRB (Interactive Ruby), вы можете использовать редирект вывода командной оболочки.

Запустите IRB:
 
$ irb
Откройте файл для вывода. Вы можете выбрать
 собственное имя файла и режим записи (например, "w" или "a" для перезаписи или добавления в конец файла):
 
output_file = File.open("output.txt", "w")
Используйте следующую команду для перенаправления вывода в файл:
 
$stdout = output_file
Теперь все выводимые данные в IRB будут записаны в файл "output.txt".

Протестируйте, чтобы убедиться, что вывод идет в файл:
 
puts "Привет, мир!"
После того, как вы закончите выводить в файл,
 не забудьте вернуть стандартный вывод обратно на консоль, чтобы прервать перенаправление:
 
$stdout = STDOUT
Закройте файл, чтобы завершить:
 
output_file.close
На этом этапе весь вывод IRB возвращается на консоль, а данные записываются в файл "output.txt".

Вот полный пример кода:

 
output_file = File.open("output.txt", "w")
$stdout = output_file

puts "Привет, мир!"

$stdout = STDOUT
output_file.close
Теперь в файле "output.txt" будет записано "Привет, мир!".

Обратите внимание, что перенаправление вывода работает
 только для команды puts. Если вы используете другие
 методы вывода (например, p, print, pp), вам нужно явно перенаправлять вывод для каждого из них.
